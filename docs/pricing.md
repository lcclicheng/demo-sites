> 配套文档：`docs/delivery-handover.md`（交付标准化包）、`docs/workflow.md` §5 / §13
> **状态**：本页为**初始报价基准**。首个客户交付完成后将基于实际工时与客户反馈**复盘重定**（见文末「定价复盘约定」）。

## 货币与税务

- 报价货币：**英镑（£ GBP）**，面向英国小商家。
- 价格**不含 VAT**（英国 VAT 注册阈值 £85,000/年；未达阈值不强制收 VAT，但报价标注 ex VAT 是专业惯例）。如未来注册 VAT 另议。
- 建站费为一次性；年度支持按年续费。

## 三档套餐

### Tier 1 — Starter（基础建站）£490

适合：想要一个干净、专业、能上线接客的静态站、内容不常变的小商家。

包含：

- 1 个高端静态站点（基于 8 套行业模板之一，Vite + React 18 + TS + Tailwind）
- 部署到 GitHub Pages（零服务器成本），含 SEO 基础（og: / sitemap / robots）、隐私政策页自动注入
- 交付标准化包（`docs/delivery-handover.md`）：欢迎邮件、合规交付清单签字版、站点维护手册（一页纸）、支持条款
- **1 轮内容修订**（交付前）
- ❌ 不含自助 CMS：后续改内容由你（owner）代改并重新部署（见 §5.9 维护流程）

### Tier 2 — CMS Enable（+CMS 自助启用）£790

在 Starter 基础上增加：

- Decap CMS 自助后台（`/admin`），客户可自行改菜单 / 文案 / 评价 / 营业时间 / 图片路径，保存即自动重建上线
- CMS 一页纸上手卡（客户如何改内容）
- **上线后 7 天**微信 / 邮件培训支持（见 `docs/workflow.md` §5.10 反馈循环）

> 差价值：把「每次改内容都找你」变成「客户自助」，显著降低你的长期运维负担。

### Tier 3 — Annual Care（年度支持）£1,290 / 年

在 CMS Enable 基础上增加（按年续费）：

- 站点健康监控（UptimeRobot + GitHub Issues 看板，异常主动告警）
- **优先响应**：支持请求 48 小时内响应（具体 SLA 见支持条款）
- **免费小改动额度**：含合理频次的小改动（如改一段文案 / 加一项菜单 / 换一张图）；额度与频次在支持条款中明确，首客后复盘确认
- 年度域名 / DNS 续费协助（若客户用自定义域名，见 `docs/custom-domain.md`）
- 模板小幅升级（如新增板块）享折扣

- 本三档 £ 为 **2026-07-14 拍板的初始基准**，用于立刻开始接单与报价。
- **首个客户交付完成后**，基于实际投入工时、客户反馈（`docs/workflow.md` §5.10 满意度调研）、市场对标，**复盘重定**价格与套餐边界。
- 复盘结论更新本文档，并同步 `docs/workflow.md` §11 状态行。
