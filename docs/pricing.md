## 货币与税务

- 报价货币：**英镑（£ GBP）**，面向英国小商家。
- 价格**不含 VAT**（英国 VAT 注册阈值 £85,000/年；未达阈值不强制收 VAT，但报价标注 ex VAT 是专业惯例）。如未来注册 VAT 另议。
- **建站不再收一次性费用**：免费建（AI 模板 + GitHub Pages 托管，边际成本 ≈ £0），收入来自持续订阅。取消旧 £590 一次性建站与 £390/年 Care。

## 套餐（纯订阅 · 三档，免费建站作楔子）

> 结构说明：用「免费建站」作 loss-leader 楔子拿下无站商家，订阅卖声誉管理 + AI 回评 + 更新 + 基础 SEO 的持续价值。真实护城河是**声誉/AI 回评**（Starter 底座），站点只是黏性载体，不是利润中心。单位经济见 `metrics/pure-subscription-model.md`。

### Free — 3 条评价回复草稿（£0，冷钩子）

适合：全体英国本地商户（不分有无官网），作首触冷外联钩子。

包含：
- 3 条真实 Google 评价 AI 回复草稿（可复制粘贴，真能用）。
- 目的：零风险拿下客户与现金流，建立信任。

### Starter — 声誉管理（£29/月）

适合：已有官网、只需把「评价有人回、评分在涨」外包出去的商户（Track B 主落档）。

包含：
- 每周 AI 评价回复稿（商户 GBP 账号发出，代运营不冒充）。
- 新评提醒 + 基础声誉监测。
- 目的：低门槛入门、建月度现金流。

### Growth — 声誉 + 免费建站（£79/月）★ 主利润档 / Track A 无站落此档

适合：无官网、想要一个专业可上线站点 + 声誉托管的商户（Track A 主落档）。把「先付 £590 建站」变成「**站免费建，你只管月付订阅**」。

包含（在 Starter 全部基础上，额外）：
- **免费高端静态站点**（基于行业模板，Vite + React 18 + TS + Tailwind），部署到 GitHub Pages（零服务器成本）。
- 站点内容更新（文案 / 图片 / 菜单 / 营业时间，邮件提需求，由你经 GitHub Pages 重建上线）。
- 基础 SEO（og: / sitemap / robots、隐私政策页自动注入）。
- 交付标准化包（`docs/delivery-handover.md`）：欢迎邮件、合规交付清单、站点维护手册（一页纸）、上手卡、支持条款、账单。
- 上线后 7 天邮件 / 微信支持（见 `docs/workflow.md` §5.10 反馈循环）。

> **价值锚**：旧模式首年 £980（£590 建 + £390 护，且先卡 £590 门槛）；新模式 **£79/mo = £948/年**，零 upfront 阻力、recurring、同等留存下 2–3× 转化率放大总营收。详见 `metrics/pure-subscription-model.md`。

> **首月后改动归口**：所有内容修改由你经 GitHub Pages 重建上线（可控、可回滚），不交付客户自助后台——省去为每个客户配 OAuth / 培训 / 维护后台的隐性成本。

### 年付优惠（可选 · 压 churn）

- **Growth 年付**：£79 × 12 = £948，年付立减 2 月 → **£790/年**（≈ £66/mo 等效），鼓励锁定。
- **Starter 年付**：£29 × 12 = £348 → **£290/年**（≈ £24/mo 等效）。
- 年付不适用退款（已享折扣），但可随时降级到 Starter。

## 上升阶梯（楔子 → 订阅，核心模式）

```
免费评价样例(£0)  →  £29/月 Starter（信任 + 现金流）
        │  用 1–2 个月、看到评分/客流改善、信任已建立
        ▼
£79/月 Growth（含免费建站 + 托管 + 更新 + 基础 SEO，无站商家顺理成章落此档）
```

客户先花小钱（或零钱）解决当下痛点，验证你靠谱，再放心把网站 + 声誉都交给你——转化率与客单价都高于冷卖高价网站。

## 合规红线（与外联 skill 一致）

- **Google 条款**：评价回复必须由**商户自己的 GBP 账号**发出，你是代运营 / 工具，不冒充批量自动化操作，避免封号。
- **GDPR**：处理顾客评价数据要有合法依据，服务条款写清；求评链接需顾客自愿。
- **AI 透明度**：回复稿可注明「AI 辅助、商户确认后发布」，避免「假装真人」争议。
- **不虚构**：外联里引用的评分 / 评价必须真实（沿用现有红线）；免费样例要真能用。
