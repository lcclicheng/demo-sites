## 发送后流程（收到回复即交 AI 替换）

1. 商家回复同意 → 发**模板 C（确认采集邮件）**（`outreach-emails.md` 末尾），收：真图 + 营业时间/电话/执业领域等核实数据 + 一句授权。
2. 把回复发我（或把图落到 `assets/<slug>/`、把数据更新进 `examples/<slug>.json`），我执行：
   - 删占位电话/编造好评 → 填真实数据
   - 真图注入 `assets/<slug>/` 并接 og:image
   - `validate-sites` → 绕守卫重建 → push（ssh 443）
3. 替换完成即消除该商家的 H1/H5 内核风险。
